# 🤖 Tasnim Chatbot

An AI-powered chatbot application built using Streamlit, LangChain, and Groq LLM API.
The chatbot provides interactive conversational responses through a simple and user-friendly web interface.

---

# 📌 Project Overview

Tasnim Chatbot is a lightweight conversational AI application that allows users to interact with a Large Language Model (LLM) in real time. The project demonstrates the integration of:

* Streamlit for frontend UI
* LangChain for LLM orchestration
* Groq API for fast AI inference
* Session state management for chat history

This project showcases practical skills in AI application development, prompt engineering, and web-based chatbot deployment.

---

# 🚀 Features

* Interactive chat interface
* Real-time AI-generated responses
* Conversation history support
* Clean and responsive Streamlit UI

# 💻 Core Functionalities

## 1. User Input Handling

Users can interact with the chatbot through Streamlit's chat interface.

## 2. Conversation Memory

Session state is used to store conversation history during runtime.

## 3. Prompt Engineering

LangChain prompt templates structure the interaction between user and AI model.

## 4. AI Response Generation

The Groq LLM processes user prompts and generates responses in real time.
